## 🌟 Recursos

- **Explorar Filmes:** Veja os filmes populares, mais bem avaliados e os últimos lançamentos em cartaz.
- **Busca Detalhada:** Encontre qualquer filme através do mecanismo de busca inteligente.
- **Favoritos e Assistidos:** Gerencie seus filmes assistidos diretamente pelo aplicativo.
- **Temas (Modo Escuro):** Suporte nativo para Modo Claro e Modo Escuro, de acordo com a preferência do seu celular.
- **Integração com TMDB:** Dados sincronizados em tempo real utilizando a robusta API do *The Movie Database*.

## 🚀 Tecnologias Utilizadas

Este projeto foi construído utilizando as melhores ferramentas do ecossistema mobile atual:
- **React Native** + **Expo**
- **TypeScript** para tipagem forte e maior segurança.
- **React Navigation** (Navegação fluída por abas e empilhamento).
- **Axios** para requisições de API.

## 📦 Como rodar o projeto localmente

Siga os passos abaixo para rodar o projeto no seu ambiente de desenvolvimento:

1. **Clone o repositório:**
   ```bash
   git clone https://github.com/DiaMont30/cine-app.git
   cd cine-app
   ```

2. **Instale as dependências:**
   ```bash
   npm install
   ```

3. **Configure as Variáveis de Ambiente:**
   Crie um arquivo `.env` na pasta raiz do projeto e insira seu token de leitura do TMDB:
   ```env
   EXPO_PUBLIC_TMDB_TOKEN=seu_token_aqui
   ```

4. **Inicie o servidor do Expo:**
   ```bash
   npx expo start
   ```
