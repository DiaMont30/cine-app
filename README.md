# CineApp 🎬

Bem-vindo ao **CineApp**, um aplicativo mobile desenvolvido em React Native que permite aos usuários descobrir, buscar e favoritar filmes!

## 🚀 Tecnologias Utilizadas

Este projeto foi construído utilizando as seguintes tecnologias e bibliotecas:

- **[React Native](https://reactnative.dev/)**
- **[Expo](https://expo.dev/)**
- **[TypeScript](https://www.typescriptlang.org/)**
- **[React Navigation](https://reactnavigation.org/)** (Navegação Stack e Tabs)
- Consumo de dados utilizando a API do **TMDB (The Movie Database)**

## 📦 Como rodar o projeto localmente

Siga os passos abaixo para rodar o projeto no seu ambiente de desenvolvimento:

1. Clone o repositório:
   ```bash
   git clone https://github.com/DiaMont30/cine-app.git
   ```

2. Acesse a pasta do projeto:
   ```bash
   cd cine-app
   ```

3. Instale as dependências:
   ```bash
   npm install
   ```

4. Inicie o servidor do Expo:
   ```bash
   npx expo start
   ```
